ModelReviewTool will be useful to manual review the Simulink model and to generate
the review report in html/xls/ppt formats.
 
Syntax: To launch the tool, add the entire ModelReviewTool folder into MATLAB path. 
Then, type the following command in the MATLAB command window
>> ModelReviewTool

Detailed help can be found from the following document "docs/MoelReviewTool_doc.pdf".
